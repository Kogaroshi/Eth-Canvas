This project is inspired by the Million Dollar Homepage.
This canvas is composed of 1 million pixels, each having a color.
Each pixel is a NFT (PXL), so they can be bought and sold. It's also possible to change the color of your Pixels.




This project was made with the purpose to understand the implementation of ERC721 Tokens, and improve my Solidty skills.
